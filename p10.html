HTML LIST

There are 3 types of list.

1) Order or number list (Using OL tag)

2) Unordered or Bullet list (Using UL tag)

3) Description list (Using DL tag)

Order or Numbered list Types

Numeric Number (1, 2, 3)

Captical Roman Number (I, II, III)

small roman number (i, ii, iii)

Capital alphabet(A,B,C)

Small alphabet(a,b,c)
